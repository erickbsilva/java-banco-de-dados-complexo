# Loja - Projeto de Banco de Dados Complexo

## 📌 Sobre
Projeto de exemplo usando JPA/Hibernate para gerenciar pedidos, clientes e produtos.

## 🚀 Como executar
1. Clone o repositório
2. Configure o banco de dados no arquivo `persistence.xml`
3. Rode `mvn clean install`
4. Execute a classe principal em `br.com.alura.loja.testes`

## 🛠 Tecnologias
- Java 11
- Maven
- JPA / Hibernate
- H2 Database

## 📂 Estrutura
- `modelo/` → entidades
- `dao/` → acesso a dados
- `dto/` → objetos de transferência
- `testes/` → classes de teste
